Ventilation App.

School group project.

The main purpose of this project was to create an application that could control a ventilation system in a classroom based on the user's emotions. The ventilation system has multiple zones and each zone can be controlled independently of the other zones. By customizing the indoor climate to the users' needs, their overall comfortability would increase and also their ability to concentrate better.

This application were made using Angular and Ionic.



The app is based on a majority voting system, in which the user first declares if they agree or disagree with the conditions presented. If the user wants a condition to change, they must choose to what degree they want it to change via a slider. An example is that the app asks if the user is cold, they agree and now they must choose within a range from 1-10 how cold they are.

The way the ventilation changes is that the app first registers how many agreed or disagreed to the condition, and then the majority decides what is to happen. To get the exact setting for the ventilator, the average number from what the users’ have answered in the slider is calculated. In each ventilation zone you can answer these questions and therefore the zones work independently of each other. The app is also time based so the ventilator does not change every five minutes because then the indoor climate would not really change. The timer is set so that the user can answer the questions every break and this way the app will not take focus away from the classes that users are attending.

The solution also has a teaching element where the users can learn different techniques to maintain a healthy indoor climate so that they can be more focused and concentrate on their studies.

In this project I had a lot of focus on the development of the design and the frotend of the application. I made all the figures for the app. The figures were used to help guide the user as to what they should do. They also brought in some color and made the app seem a bit more fun. For the frontend part, I coded the main part of the app where you choose to change the ventialtion. For this I used the frameworks ionic and anglar and a lot of cosutm css.
